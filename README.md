# Blade-Redis

在java中更方便的操作redis数据库，基于`blade`框架继承，支持多数据库集群配置。

##使用

```java
// 配置
RedisPlugin redisPlugin = RedisPlugin.me();
redisPlugin.host("127.0.0.1", 6379);
redisPlugin.run();
```
